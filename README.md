# ITQ 인터넷 자동채점 시스템

아카데미소프트 ITQ 인터넷 자격증 수강생 답안 제출 및 자동채점 웹사이트입니다.

## 주요 기능

- 수강생 이름 선택 후 모의고사 / 기출복원 답안 제출
- 제출 즉시 자동채점 (총 500점 기준)
- 유형별 점수 및 레이더 차트 시각화
- 누적 성적 데이터 저장 (Firebase)
- 강사 전용 관리 화면 (수강생 추가/삭제, 전체 성적 조회)

## 시험 구성

| 유형 | 내용 | 배점 |
|------|------|------|
| 유형1 | 인터넷 윤리 | 60점 |
| 유형2 | 인터넷 검색 - 일반검색 I | 30점 |
| 유형3 | 인터넷 검색 - 일반검색 II | 100점 |
| 유형4 | 지능형 정보검색 | 90점 |
| 유형5 | 인터넷 검색 - 실용검색 | 150점 |
| 유형6 | 정보 가공 | 70점 |

## 합격 기준

| 등급 | 점수 |
|------|------|
| A등급 | 400점 이상 |
| B등급 | 300 ~ 399점 |
| C등급 | 200 ~ 299점 |
| 불합격 | 200점 미만 |

## 사용 방법

1. 메인 화면에서 본인 이름 선택
2. 시험 유형(모의고사/기출복원)과 회차 선택
3. 답안 입력 후 채점하기
4. 결과 화면에서 점수 및 유형별 분석 확인

## 강사 관리

- 메인 화면 하단 **강사 관리 화면** 접속
- 비밀번호 입력 후 수강생 추가/삭제 및 전체 성적 조회 가능

## 기술 스택

- Frontend: HTML / CSS / JavaScript
- Database: Firebase Firestore (무료 Spark 플랜)
- Hosting: GitHub Pages
- Chart: Chart.js
